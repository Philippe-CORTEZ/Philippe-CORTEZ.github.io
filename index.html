Qui suis-je ?

Bonjour et bienvenue sur mon site web personnel.

Avant tout, je tiens à préciser que ce site ne fait pas office de CV. Il est plutôt un moyen d'en apprendre plus sur moi, mon parcours, mes accomplissements, mes passions ! :D

Moi c'est Philippe CORTEZ, je suis actuellement étudiant en informatique et un passioné par le domaine.
Pour commencer, j'aimerais vous raconter dans cette rubrique, comment j'ai découvert l'informatique:

C'est à environ l'âge de 15ans que j'ai eu pour la première fois un ordinateur entre les mains (un ASUS X53S blanc, que de souvenirs !).
Un des collègues de travail de mon père était lui aussi un passioné d'informatique, c'est à ses coté que j'ai pu avoir une introduction en la matière ;)

Je me suis très vite intéressé dans mon coin (comme beaucoup de jeunes) à la cybersécurité et au hacking de manière plus générale.
Comme premier fait d'armes, je pourrais vous citer le jailbreak de la console de l'époque, la Wii !
C'est dans un forum d'entraide au jailbreak de la console, que j'ai pu rencontrer d'autres personnes formidable avec qui j'ai pu découvrir le monde du développement.

Les années passées, me voila maintenant dans ma dernière année d'étude en informatique, en master.
J'aime toucher à tous les domaines de l'informatique, notamment la cybersécurité, le développement et le monde de la data.

Mes projets

Voici une liste non exhaustive des projets que j'ai pu réaliser

Animalia

Animalia est une application de gestion qui est utile pour les refuges d'animaux

L'application permet d'enregistrer un nouveau pensionnaire, adoptant, de consulter les données d'un pensionnaire
Notamment son refuge actuel, sa fiche médicale (pathologies et soins), et bien d'autre.

L'application gère aussi tout le processus d'adoption d'un pensionnaire. A terme elle devrait aussi permettre le changement d'un refuge et la mise à jour des informations médicales d'un pensionnaire.

Animalia est un projet pour mettre en pratique certaines technologies vues en cours notamment les API REST, docker, Les bases de données relationnelles et JPA via une ORM comme eclipselink.
Une application lourde côté client avec JavaFX. Les différents designs patterns comme MVC, DAO

Dépôt github

Wide Factory

Wide Factory est une application d'aide à la pratique physique et sportive.

L'application permet de prévenir les risques de blessure lors d'une activité physique.
Elle convient à toute personne voulant faire du sport en évitant de se blesser, que vous soyez un particulier ou un professionnel.

Wide Factory utilise pour la détection de mouvements une Microsft Kinect Azur dont il est necéssaire d'avoir.

Ce projet était un projet scolaire en master 1, en groupe composé de 7 personnes, avec une méthologie SCRUM dont je fus le SCRUM master.
Coté technique, le projet mobilise les connaissances suivantes :
Pour le client : JAVA, websocket, JPA, JavaFX, JMonkey Engine.
Pour le serveur : C++, Azure Kinect C++ API, uwebsockets, BDD postgreSQL.

Si vous êtes intéressé par le projet, allez visiter les dépots github.

Hyerplanning

HyperPlanning est un projet de groupe de 4 personnes en master 1, dont je fus le SCRUM master.
Le but étant de mettre à profit les connaissances en JAVA et les différents design pattern vu en cours.

Le projet est simple et imite la vraie application hyperplanning, c'est-à-dire, un gestionnaire de planning.
Le développement de ce projet à été répartit en 3 semaines avec des intervenants de l'entreprise Sopra Steria.

Les super fourmis

Les super fourmis est un projet scolaire de L3
Le projet met en oeuvre du développement d'IHM en python (avec Tkinter) et l'analyse UML.

Le projet simule la vie de colonies de fourmis cherchant de la nourriture (des ressources).
L'application est interactive, vous pouvez poser des ressources, des fourmilières et des obstacles de plusieurs natures (murs, feu, ...)
Lancer ensuite la simulation pour voir les fourmis se déplacer.

Canon!

Canon! est un petit projet scolaire de L3 à l'occasion d'un module sur la géométrie algorithmique.

Le projet permet de mettre en oeuvre les connaissances du cours avec pyOpenGL (opengl pour python).
Le but est simple, vous êtes un canon que vous pouvez déplacer sur une berge, une rivière vous sépare d'une cible sur l'autre berge
Visez et tirez pour détruire la cible !

Cube isométrique

Cube isométrique est un projet scolaire de L3, mettant en oeuvre les enseignements en POO python et IHM (Tkinter).

L'application consiste à construire des scènes en vue isométrique en posant des cubes, à la hauteur, position et couleur voulue.

Compilateur machine RAM

Le projet Compilateur machine RAM (Register Adressable Memory) est un projet scolaire de L3 à l'occasion du module sur la théorie des langages.

Le projet compile du code écrit en pseudo code vers un code pour le modèle de calcul de la machine RAM (assez proche de l'assembleur).

Écrit en C avec les librairies Flex et Bison.
